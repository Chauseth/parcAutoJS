POUR ACCEDER A LA CONSOLE :

Nom de la variable de test : maVoiture

Nom des différentes fonctions :

  • repeindre(changecolor) -> Permet de changer la couleur du véhicule
  • mettreEssence(addedFuel) -> Permet d'ajouter de l'essence dans la limite du réservoir
  • seDeplacer(distance, medianSpeed) -> Permet de simuler un déplacement et indique le carburant restant
  • consumptionCalcul(distance, consumption) -> Sert pour la fonction seDeplacer et permet de calculer la consommation
  • setInsurance() -> Permet de changer la valeur de l'assurance
  • toString() -> Permet de récupérer les infos de la voiture
